# Zemer Ressources International 
Sites vitrine officiel de Zemer Ressources International.
## Stack technique
-Frontend : HTML / CSS / JS
-Serveur  : Nginx (Alpine) 
-Conteneur: Docker + Docker Compose 
-CI/CD    : GitHub Actions 
##Lancer le projet 
### Avec Docker
docker-compose up -d
### Accès local
http://localhost : 8080
## Structure 
html/       Code frontend
nginx/      Config serveur
docs/       Documentation
